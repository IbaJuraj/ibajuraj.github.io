🔒 Ochrana súkromia

Súkromie používateľov je pre nás jednou z najvyšších priorít.

Aplikácie IbaJuraj Apps nepredávajú osobné údaje tretím stranám.

Údaje zostávajú uložené predovšetkým v zariadení používateľa, pokiaľ nie je pri konkrétnej funkcii uvedené inak.

Naším cieľom je ponúknuť aplikácie, ktoré rešpektujú súkromie a dávajú používateľovi plnú kontrolu nad jeho údajmi.
